La intersección de dos conjuntos A y B es el conjunto formado por los elementos comunes a los dos conjuntos. Se define como:

A ∩ B = { x | x ∈ A y x ∈ B}

En la intersección de conjuntos se verifican las siguientes propiedades:

i) Un conjunto intersecado consigo mismo es el mismo conjunto.

A ∩ A = A

ii) Un conjunto intersecado con el conjunto vacío es igual al conjunto vacío. A ∩ ∅ = ∅

iii) El conjunto A intersecado con el conjunto universo es el conjunto A.

A ∩ U = A

La unión de dos conjuntos A y B es el conjunto formado por la agrupación de todos los elementos de A y todos los elementos de B. Se define como: A ∪ B = { x | x ∈ A o x ∈ B}.

En la unión de conjuntos se verifican las siguientes propiedades:

i) Un conjunto unido consigo mismo es el mismo conjunto. A ∪ A = A

ii) Un conjunto unido con el conjunto vacío es igual al mismo conjunto. A ∪ ∅ = A

iii) El conjunto A unido con el conjunto universo es el conjunto universo. A ∪ U = U

La diferencia de dos conjuntos A y B es el conjunto formado por los elementos de A que no pertenecen a B. Se define como: A − B = { x | x ∈ A y x ∉ B}.

En la diferencia de conjuntos se pueden establecer las siguientes propiedades:

i) La diferencia del conjunto A consigo mismo es el conjunto vacío. A − A = ∅

ii) La diferencia de dos conjuntos disjuntos A y B está formada solo por los elementos del primer conjunto. Si A y B son disjuntos, entonces A − B = A .

iii) La diferencia del conjunto A con el conjunto vacío es el conjunto A. A − ∅ = A

La diferencia simétrica de dos conjuntos A y B es el conjunto formado por los elementos de la unión de (A – B) y (B – A). En forma simbólica se representa así: A Δ B = { x | x ∈ (A − B)  ∪ ​(B − A) }.

También se puede definir a la diferencia simétrica como: A Δ B = { x | x ∈ (A ∪ B) y x ∉ (A ∩ B) }.

El complemento de un conjunto A es la diferencia entre el conjunto universo U y dicho conjunto. Se simboliza así: A' = U − A
